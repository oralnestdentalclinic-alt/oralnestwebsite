# OralNest Dental Clinic — Enterprise-Grade Patient Experience 🦷 

OralNest is a high-fidelity, full-stack web platform engineered for a premium dental clinic in Wakad, Pune.  
The project blends clinical precision with a refined digital experience, catering to both local patients and the growing international dental tourism market.

🌐 **Live Site:** https://oral-nest-wakad.vercel.app

---

## ✨ Overview

This project goes beyond a conventional website. It is a **digitally engineered patient journey**, designed to deliver trust, clarity, and ease from the very first interaction.

Every design and technical decision was driven by real-world usability, ensuring the platform performs seamlessly in a live clinical environment.

---

## 💡 Key Highlights

- Structured **patient onboarding flow** to minimize friction  
- Real-world **clinical workflow integration** into UI/UX  
- Built for **production deployment**, not just demonstration  
- Strong emphasis on **performance, SEO, and conversions**  
- Clean, minimal, and premium design language  

---

## 🧠 Engineering Approach

### Stakeholder Collaboration
Worked directly with clinic founders to understand operational workflows and patient expectations.  
This led to the creation of a tailored **“Discreet & Structured Care Journey.”**

### Problem-First Development
Focused on identifying inefficiencies in patient intake and solving them through a **centralized and reusable form system**.

### Real-World Deployment
Designed to handle:
- Live user traffic  
- Consistent brand experience  
- Reliable lead generation  

---

## ⚙️ Features

- ✨ Smooth animations using Framer Motion  
- 🧾 Unified lead capture system (Web3Forms integration)  
- 🖼️ Interactive before/after transformation slider  
- 📱 Fully responsive design across all devices  
- ⚡ Optimized performance with server-side rendering  
- 🎯 Conversion-focused UI structure  

---

## 🛠️ Tech Stack

- **Framework:** Next.js 14 (App Router)  
- **Styling:** Tailwind CSS  
- **Frontend Logic:** React Hooks  
- **Animations:** Framer Motion  
- **Forms API:** Web3Forms  
- **Deployment:** Vercel Edge Network
